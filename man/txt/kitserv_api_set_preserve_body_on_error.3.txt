kitserv_api_set_preserve_body_on_error(3)

SYNOPSIS
     library "kitserv"

     #include <kitserv.h>

     void
     kitserv_api_set_preserve_body_on_error(struct kitserv_client*,
	 bool preserve_enabled);

DESCRIPTION
     The kitserv_api_set_preserve_body_on_error() function instructs Kitserv
     to not discard the body or headers when returning an error response to
     the client.

     For more information, see kitserv_api_set_preseve_headers_on_error().
