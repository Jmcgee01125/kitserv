kitserv_api_reset_headers(3)

SYNOPSIS
     library "kitserv"

     #include <kitserv.h>

     void
     kitserv_api_reset_headers(struct kitserv_client*);

DESCRIPTION
     The kitserv_api_reset_headers() function clears the headers of a re-
     sponse.
