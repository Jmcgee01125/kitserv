kitserv_api_get_request_content_length(3)

SYNOPSIS
     library "kitserv"

     #include <kitserv.h>

     off_t
     kitserv_api_get_request_content_length(struct kitserv_client*);

DESCRIPTION
     The kitserv_api_get_request_content_length() this function returns the
     content length of the client's request. If the client did not provide a
     content-length header, this function returns 0.
