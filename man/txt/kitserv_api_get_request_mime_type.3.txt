kitserv_api_get_request_mime_type(3)

SYNOPSIS
     library "kitserv"

     #include <kitserv.h>

     const char*
     kitserv_api_get_request_mime_type(struct kitserv_client*);

DESCRIPTION
     The kitserv_api_get_request_mime_type() function returns a pointer to a
     null-terminated string for the client's value in the content-type header.
