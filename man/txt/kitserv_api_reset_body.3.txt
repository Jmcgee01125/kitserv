kitserv_api_reset_body(3)

SYNOPSIS
     library "kitserv"

     #include <kitserv.h>

     int
     kitserv_api_reset_body(struct kitserv_client*);

DESCRIPTION
     The kitserv_api_reset_body() function clears the response body buffer.

     Note: This does not affect range or file information.
